# Contributing to Awesome Greek Computing Jobs

Thank you for wanting to improve this list! We love community contributions. To keep the repository organized, please follow these simple steps to add or update a company.

## How to add a row via Pull Request

1. **Fork the Repository**: Click the 'Fork' button at the top right of the main page.
2. **Edit the README**: Open the `README.md` file in your fork.
3. **Add your row**:
   - Locate the **"Engineering Hubs & Career Portals"** table.
   - Add a new line using the following format:
     `| **Company Name** | Sector | [Careers Page](URL) | [LinkedIn](URL) |`
   - Make sure to keep the alphabetical order if possible!
4. **Commit Changes**: Use a clear commit message like `feat: add [Company Name] to engineering hubs`.
5. **Create Pull Request**: Go back to the original repository and click "New Pull Request".

## Contribution Rules
* **Tech Focus Only**: Please only add companies hiring for Computer/Software Engineering, Data, or Tech-Business roles. No mechanical, civil, or non-tech engineering.
* **Working Links**: Ensure both the Careers Page and LinkedIn Profile links are active and correct.
* **Formatting**: Maintain the table structure (the `|` pipes) so the README stays readable.
